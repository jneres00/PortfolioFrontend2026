Projetos

Projetos desenvolvidos durante minha jornada de aprendizado em front-end.

APP To-do List
Projeto de aplicativo de tarefas com tema gamer. Permite adicionar, concluir e excluir tarefas do dia a dia. O HTML foi adaptado a partir de aulas no YouTube, enquanto o design foi totalmente criado por mim, inspirado em um protótipo feito no Canva.

Guia de cuidados de pets
Pequeno guia interativo sobre cuidados com diferentes pets, com links para referência (ex.: Wikihow) e barra de pesquisa. Projeto responsivo desenvolvido com Bootstrap, focando em clareza e praticidade para o usuário.

Formulário
Página de formulário para inscrição em um site de receitas, com interface intuitiva e moderna. Os usuários podem receber receitas exclusivas diretamente no e-mail, com destaque para usabilidade e estética visual agradável.

Calculadora de Desconto de Compras
Uma calculadora web que aplica automaticamente 10% de desconto em itens de mercado e subtrai os valores de um orçamento fixo de R$500, mostrando o saldo restante. O projeto foi pensado para ser simples e intuitivo.

Biblioteca Virtual
Uma biblioteca online criada com HTML, CSS e Bootstrap. O site apresenta um livro em destaque e um catálogo organizado por categorias, como romance e infantil. Cada livro possui um botão que direciona o usuário para a página de compra na Amazon.

Fanpage do Joji
Uma fanpage dedicada ao meu cantor favorito. O site inclui uma galeria de fotos interativa, uma página com a biografia do artista, links para redes sociais e um player do Spotify para ouvir músicas diretamente no site.

Fanpage de Anime
Uma página dedicada ao anime que teve grande significado para mim. O projeto foi criado para praticar estrutura de páginas e organização de conteúdo.
